# SharpBoy Opcode Manual

This is a React/Vite port of https://meganesu.github.io/generate-gb-opcodes/ intended for iterating on features like searching, scrolling, and responsive layout.

## Credits

This project is based on the most excellent original work:

- https://github.com/meganesu/generate-gb-opcodes

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
